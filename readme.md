# Blog project with Strapi CMS
- Styles with Tailwind CSS

## Run Project Blog Project
- `git clone https://github.com/gabrielcarlossl/strapi-blog.git`

- `npm i`

- `npm run dev`

- `http://localhost:5173/`

## Run Strapi CMS

- `cd strapi`

- `npm i`

- `npm run develop`

- `http://localhost:1337/admin/auth/login`
